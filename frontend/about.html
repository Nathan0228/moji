关于墨迹

墨迹 Ink Trace

墨迹（Ink Trace）是一款专注于书法练习与成长记录的轻量化平台。 用户可以上传每日练习作品，记录学习过程， 并通过成长时间轴与热力图回顾自己的书法历程。

设计理念

每一次落笔都值得被记录。 书法学习并非一蹴而就，而是长期积累的过程。 墨迹希望帮助书法爱好者建立持续记录与反思的习惯， 用数字化方式保存成长轨迹。

功能特色

• 作品上传与展示
• 成长时间轴记录
• GitHub 风格学习热力图
• 农历日期与传统纪年展示
• 图片缩放与全屏预览
• 用户个人成长档案

技术栈

Frontend：HTML5 / CSS3 / JavaScript
Backend：Go
Database：MySQL
Authentication：JWT
