# 墨迹 Ink Trace

一个专注于书法学习与成长记录的轻量化平台。

## 项目简介

墨迹（Ink Trace）旨在帮助书法爱好者记录每日练习成果，通过数字化方式保存和展示书法成长轨迹。

用户可以上传作品、查看成长时间轴、浏览年度热力图，并使用传统农历日期展示作品创作时间。
